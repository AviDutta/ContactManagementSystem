# ContactManagementSystem
	
This application’s main purpose of maintaining contact information by providing CRUD operations such as create, update and delete on it.

Live website URL:- http://riteshskoshti-001-site1.itempurl.com/


## Technologies used for development
   *ASP.NET MVC
   *REST API (WEB API)
   *SQL Server 2012
   *Microsoft Unit Test framework
   
   
## Projects in Solution
1. ASP.NET MVC:- ContactManagementWebSite
2. REST API:- ContactManagementServices
3. Unit Test:- ContactManagementWebSiteUnitTest
